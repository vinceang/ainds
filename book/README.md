# The AINDS Handbook

**Version:** 0.1 Draft  
**Working Title:** *The AI-Native Design Systems Handbook*  
**Tagline:** Knowledge Before Generation

AINDS is not only a codebase or component library. It is a methodology for transforming tribal knowledge into organizational knowledge so humans and AI can create consistent, accessible, scalable digital products from a shared source of truth.

## Editorial Rule

Every chapter should end with:

1. **Principles**
2. **Implementation Guidance**
3. **Knowledge Before Generation**
