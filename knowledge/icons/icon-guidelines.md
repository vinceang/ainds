# Icon Guidelines

AINDS icons should feel like one family.

## Default Style

- 24px viewBox
- 2px stroke
- Rounded line caps
- Rounded joins
- No unnecessary fills
- Optical alignment over mathematical alignment
- Consistent visual weight
- Simple geometry
- Recognizable silhouettes

## AI Icon Specialist Must Know

Before creating a new icon, the AI should review:

- Existing icons
- Stroke rules
- Grid rules
- Naming conventions
- Export requirements
- Intended usage
