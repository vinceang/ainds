# AINDS

**AI-Native Design Systems**

> Knowledge Before Generation.

AINDS is a methodology and knowledge architecture for building design systems where AI is a first-class participant in product development.

The goal is not to let AI invent products from scratch. The goal is to capture design, engineering, accessibility, UX, governance, and product knowledge so humans and AI can generate consistent, explainable, accessible artifacts from a shared source of truth.

## Core Idea

Most AI tools ask:

> What should I generate?

AINDS asks:

> What should I know before I generate?

## Repository Purpose

This repository is the initial knowledge base for AINDS. It captures the founding philosophy, ADRs, principles, agent definitions, governance model, and reference architecture.

## North Star

**Knowledge Before Generation**

AI should not generate from assumption. AI should generate from documented knowledge.
