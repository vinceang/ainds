# Glossary

## AINDS

AI-Native Design Systems.

A methodology for building design systems where AI is a first-class participant in product development.

## Knowledge Before Generation

The central principle of AINDS. AI should generate from documented knowledge, not assumption.

## AI Specialist

A focused AI role responsible for a specific product-development capability, such as component creation, accessibility review, icon generation, or Storybook documentation.

## Knowledge Base

The documented source of truth that describes design principles, tokens, components, accessibility standards, UX patterns, and governance.

## Reference Implementation

A working implementation of the AINDS methodology, likely including tokens, Web Components, Angular adapters, Storybook, and example products.
