# Foundational Principles

## 1. Knowledge Before Generation

AI must understand before it creates.

## 2. Intent Before Implementation

Humans define intent. AI accelerates implementation.

## 3. Systems Before Screens

AINDS prioritizes reusable systems over one-off screens.

## 4. Accessibility by Default

Accessibility is a requirement, not a later audit.

## 5. Documentation Is the Product

If it is not documented, it is not finished.

## 6. Platform Over Framework

Knowledge should outlive any framework implementation.

## 7. Humans Own Intent

AI proposes. Humans approve.

## 8. Consistency Over Novelty

AI should prioritize system consistency over unnecessary invention.
