```txt
Knowledge Base
  ├── Principles
  ├── Tokens
  ├── Components
  ├── UX Patterns
  ├── Accessibility Standards
  ├── Icon Guidelines
  ├── Illustration Guidelines
  ├── Content Guidelines
  ├── i18n Rules
  └── ADRs
        │
        ▼
AI Specialists
  ├── Design System Architect
  ├── Component Engineer
  ├── Storybook Author
  ├── Accessibility Auditor
  ├── UX Writer
  ├── Icon Specialist
  ├── Illustration Specialist
  ├── Localization Specialist
  └── Product Bootstrapper
        │
        ▼
Reference Implementation
  ├── Tokens
  ├── Web Components
  ├── Angular Adapters
  ├── Storybook
  └── Example Products
```

## Guiding Architecture Rule

The knowledge base is the source of truth.

Components, agents, documentation, and products are consumers.
