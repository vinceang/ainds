# Sessions

This folder captures working-session notes.

The conversation is discovery.  
The repository is memory.

## Rule

Nothing important lives only in chat.

Each meaningful session should produce a session note that captures:

- Date
- Summary
- Decisions
- New principles
- New terms
- ADR candidates
- Book updates
- Knowledge base updates
- Open questions
- Next actions

Session notes are raw-but-structured field notes. Important items should later be promoted into permanent artifacts.
